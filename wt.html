Tools used in web development


1-python

Python is an interpreted high-level general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant indentation. Designer: Guido van rossum
launched:Febuary 1991
what it can do:
1- Python can be used on a server to create web applications.
2- Python can be used alongside software to create workflows.
3- Python can connect to database systems. It can also read and modify files.
4- Python can be used to handle big data and perform complex mathematics.
5- Python can be used for rapid prototyping, or for production-ready software development.


2-Mysql

MySQL is an open-source relational database management system. Its name is a combination of "My", the name of co-founder Michael Widenius's daughter, and "SQL", the abbreviation for Structured Query Language.
1- Relational database
2- Open source
3- Fast,reliable,scalable and easy to use
4- Developed, distributed, and supported by Oracle Corporation


3-mongodb

MongoDB is a source-available cross-platform document-oriented database program. Classified as a NoSQL database program, MongoDB uses JSON-like documents with optional schemas.
release date:Febuary 2009
Version:1.0


4-php

PHP is a general-purpose scripting language especially suited to web development. It was originally created by Danish-Canadian programmer Rasmus Lerdorf in 1994. The PHP reference implementation is now produced by The PHP Group.
Stable release:8.0.8 / 1 July 2021
Developer:The PHP Development Team, Zend Technologies
Designed by: Rasmus Lerdorf

5-wordpress

WordPress is a free and open-source content management system written in PHP and paired with a MySQL or MariaDB database. Features include a plugin architecture and a template system, referred to within WordPress as Themes.
developer:WordPress Foundation
Stable release: 5.7.2 / 12 May 2021
Operating system:Unix-like, Windows, Linux

6-git

Git is software for tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development. Its goals include speed, data integrity, and support for distributed, non-linear workflows.
Developer:Junio Hamano
original author:Linus Torvalds

7-Opensuse

OpenSUSE is a project that serves to promote the use of free and open-source software. openSUSE is well known for its Linux distributions, mainly Tumbleweed, a tested rolling release, and Leap, a distribution with long-term support.
Developer: SUSE, Novell
Latest release:Leap 15.3/June 2, 2021

8-Cakephp

CakePHP is an open-source web framework. It follows the model–view–controller approach and is written in PHP, modeled after the concepts of Ruby on Rails, and distributed under the MIT License.
Stable release:4.2.2 / 2021-01-06
Initial release:April 2005
